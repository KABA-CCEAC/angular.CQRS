ObjectId
service in module ngCQRS

Description

Used to obtain a objectId that mimics how WCF serializes a object of type MongoDB.Bson.ObjectId and converts between that format and the standard 24 character representation. Inspired by https://github.com/justaprogrammer/ObjectId.js

Usage
mymodule.service('MyService', function(ObjectId){

var objectIdOne = ObjectId();
var objectIdTwo = ObjectId(0, 0, 0, 0x00ffffff);
var objectIdThree = ObjectId('507f1f77bcf86cd799439011');
console.log(objectIdOne.toArray());
console.log(objectIdThree.toArray());

});
