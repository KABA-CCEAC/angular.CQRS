Store
object in module ngCQRS

Description

The store allows for querying modelViews and registering for subsequent events on that viewModelName.

Methods

  • do(callback)

    register a handler for events on the specified viewModelName

    Parameters
    ParamTypeDetails
    callbackfunction

    Function that is called on first query response and on subsequent events. Angular.CQRS will pass in the denormalized object (See registerDenormalizerFunctions).

  • for(viewModelName, parameters)

    Specify the a viewModelName and optional url parameters

    Parameters
    ParamTypeDetails
    viewModelNamestring

    The identifier of the viewModelName

    parametersobject

    An optional object containing url parameters. This will be passed toghether with the viewModelName identifier into your urlFactory function.
